# navime
App for searching events near you.
## Getting Started

Based at laravel 5.5, so just run
```
composer update
php artisan serve
```

### Prerequisites

Your server must have the following requirements:

```
PHP >= 7.0.0
OpenSSL PHP Extension
PDO PHP Extension
Mbstring PHP Extension
Tokenizer PHP Extension
XML PHP Extension

```
